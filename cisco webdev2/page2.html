Chapter 1: CSS Basics

Question Answer
What does the following CSS rule do?
p { color: red; } 		It sets the text color of all < p > elements to red.
What does the following CSS rule do?
#header { font-size: 24px; } 		It sets the font size of the element with the ID header to 24 pixels.
What does the following CSS rule do?
[data-highlight] { background-color: yellow; } 		It sets the background color of all elements with the data-highlight attribute to yellow.
What does the following CSS rule do?
h1, h2, p { color: blue; } 		It sets the text color of all < h1 >, < h2 >, and < p > elements to blue.
What does the following CSS rule do?
a:focus { color: blue; }
What does the pseudo-code class selector :focus target in the given code? 		The element when it receives keyboard or mouse input focus.
What does the following CSS rule do?
[class^="button"] { color: green; } 		It sets the text color of all elements with a class name starting with "button" green.
What does the following CSS rule do?
div + p { color: blue; }
What does the adjacent sibling combinator (+) in CSS do? 		It selects an element that is directly next to a specified element at the same level in the DOM hierarchy.
In the CSS cascade, which of the following has the lowest precedence? Element selectors
In the CSS cascade, which of the following has the highest specificity? ID selectors
What is the specificity from highest to lowest? ID selectors, Class selectors, Element selectors
What effect does the !important declaration have in CSS? It overrides all the other styles regardless of specificity.
When is it appropriate to use the !important declaration? Sparingly, as a last resort to override user-agent or third-party styles.
What is the purpose of CSS comments? To provide explanations or context within the CSS code.
Which CSS property is typically inherited by child elements? font-size, font-family, color, etc.
